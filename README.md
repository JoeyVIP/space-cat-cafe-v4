# 太空貓咖啡館 V4 - Space Cat Cafe

一個充滿太空主題與可愛貓咪元素的咖啡館網站。

## 特色

- 🌌 深藍太空主題設計
- 🐱 可愛貓咪元素
- 📱 Mobile-First 響應式設計
- ✨ 現代化的 UI/UX 設計
- 🎨 動態效果與動畫

## 網站架構

- **首頁 (index.html)**: 主視覺、特色介紹、環境畫廊
- **關於我們 (about.html)**: 品牌故事、理念、太空貓咪團隊
- **菜單 (menu.html)**: 完整的飲品與餐點菜單
- **聯絡資訊 (contact.html)**: 聯絡方式、營業時間、交通資訊、聯絡表單

## 技術棧

- HTML5
- CSS3 (Mobile-First Responsive Design)
- JavaScript (Vanilla JS)
- 無外部框架依賴

## 部署

本網站已部署至 Render 靜態網站服務。

## 本地開發

1. Clone 此 repository
2. 用任何 HTTP 伺服器開啟即可，例如：
   ```bash
   python -m http.server 8000
   ```
3. 在瀏覽器開啟 `http://localhost:8000`

## 瀏覽器支援

- Chrome (最新版本)
- Firefox (最新版本)
- Safari (最新版本)
- Edge (最新版本)
